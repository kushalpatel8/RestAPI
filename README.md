# RESTful API Learning Project

A Spring Boot application designed to demonstrate the implementation of RESTful APIs using Java. This project provides a fully functional backend for managing student records with CRUD (Create, Read, Update, Delete) operations, utilizing MySQL for persistence.

* **Student Management**: Full CRUD capabilities for student entities.
* **Data Validation**: Integrated validation for incoming requests using Spring Validation.
* **DTO Mapping**: Efficient object mapping between Entities and DTOs using ModelMapper.
* **Partial Updates**: Support for `PATCH` requests to update specific fields of a resource.
* **Database Persistence**: Uses Spring Data JPA with MySQL.
